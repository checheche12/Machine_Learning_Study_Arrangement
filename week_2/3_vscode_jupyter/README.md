### VSCode와 Jupyter
 - Python은 기본적으로 `코드를 한줄씩 읽으면서 해석하는` 종류의 언어이기 때문에
 - 선행적으로 기계어로 바꾸는 `컴파일` 과정은 거칠 필요가 없습니다.
 - 그래서 `컴파일러` 는 필요하지 않지만, 좋은 텍스트 편집기가 있으면, 훨씬 작업하기 쉬워집니다.
 - 또한 학습용으로는 코드를 중간중간, 실행결과를 저장하면서 편집 할 수 있으면 편한데
 - 그런 점에서는 `Jupyter` 를 사용하면 편합니다.
 - 그렇기 때문에 VsCode 에서 Jupyter 를 사용하여, 이런 문제를 해결 할 수 있습니다.
 - 또한 `venv` 로 만든 가상환경을 Jupyter 에서 사용 가능하기 때문에 환경 문제도 압박이 없습니다.
  - 물론 Jupyter는 다른 가상환경도 모두 사용 가능합니다.
